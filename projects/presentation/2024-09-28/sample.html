01-On Mouse Down

Occured when a button is pressed down. Triggered before the button is release



02-On Mouse Up

Triggered when the mouse button is released over an element.



03-On Mouse Enter

Triggered when the mouse pointer enters the boundary of an element. It does not trigger if the pointer moves over child elements.



04-On Mouse Leave

Triggered when the mouse pointer leaves the boundary of an element. It does not trigger if the pointer moves out of child elements



05-On Mouse Move

Triggered whenever the mouse pointer moves over an element, continuously as the mouse moves.



06-On Mouse Out

Triggered when the mouse pointer leaves the boundary of an element or its child elements.



07-On Mouse Over

Triggered when the mouse pointer enters the boundary of an element or its child elements.



08-On Double Click

Fires when a user double-clicks on an element. A double-click is typically perceived as two rapid clicks, so the event fires once per double-click
